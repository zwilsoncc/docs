## Step 1: Set Up Your Svelte Project

Set up a Svelte app with [`npx`](https://www.npmjs.com/package/npx) and move into the project directory:

<Snippet dark text="npx degit sveltejs/template my-svelte-project && cd my-svelte-project" />
<Caption>Initializing a Svelte app with <Link href="https://www.npmjs.com/package/npx">npx</Link> and moving into the project directory.</Caption>

## Step 2: Deploying Your Svelte App with ZEIT Now

With a Svelte app set up, it is ready to deploy live with [Now](/docs/v2).

Using [Now CLI](/download), deploy your app with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Svelte app with Now CLI.</Caption>

If you want to deploy your Svelte app when you push to a Git repository, you can use either [Now for GitHub](/docs/v2/integrations/now-for-github/) or [Now for GitLab](/docs/v2/integrations/now-for-gitlab/) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.

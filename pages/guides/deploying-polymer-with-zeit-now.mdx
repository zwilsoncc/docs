[Polymer](https://www.polymer-project.org/) is an efficient, expressive, and extensible HTML templating library for building JavaScript applications.

## Step 1: Set Up Your Polymer Project

Set up a Polymer app with [`npx`](https://www.npmjs.com/package/npx) and move into the project directory, choosing the **blank** option when presented:

<Snippet dark text="mkdir my-polymer-app && cd my-polymer-app && npx polymer-cli init polymer-3-starter-kit" />
<Caption>Creating a project directory and initializing a Polymer app with <Link href="https://www.npmjs.com/package/npx">npx</Link>.</Caption>

## Step 2: Deploying Your Polymer App with ZEIT Now

With a Polymer app set up, it is ready to deploy live with [Now](/docs/v2).

Using [Now CLI](/download), deploy your app with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Polymer app with Now CLI.</Caption>

If you want to deploy your Polymer app when you push to a Git repository, you can use either [Now for GitHub](/docs/v2/integrations/now-for-github/) or [Now for GitLab](/docs/v2/integrations/now-for-gitlab/) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.

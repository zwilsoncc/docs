[Ember](https://emberjs.com/) is a JavaScript framework for creating ambitious web applications.

Create and deploy a new Ember project with ZEIT Now without the setup using the button below.

<a href="https://zeit.co/new/project?template=https://github.com/zeit/now-examples/tree/master/ember">
  <Image src="https://zeit.co/button" width={104} height={36} align="left" />
</a>

Alternatively, set up Ember locally and deploy using Now CLI with the following steps.

## Step 1: Set Up Your Ember Project

[Set up an Ember project](https://guides.emberjs.com/release/getting-started/quick-start/) with [`npx`](https://www.npmjs.com/package/npx) and move into the project directory:

<Snippet dark text="npx ember-cli new my-ember-project && cd my-ember-project" />
<Caption>Initializing an Ember project with <GenericLink href="https://ember-cli.com/">ember-cli</GenericLink> and moving into the project directory.</Caption>

## Step 2: Deploying Your Ember App with ZEIT Now

With an Ember app set up, it is ready to deploy live with [Now](/docs/v2).

Using [Now CLI](/download), deploy your website with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Ember app with Now CLI.</Caption>

If you want to deploy your Ember project when you push to a Git repository, you can use either [Now for GitHub](/docs/v2/integrations/now-for-github/) or [Now for GitLab](/docs/v2/integrations/now-for-gitlab/) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.

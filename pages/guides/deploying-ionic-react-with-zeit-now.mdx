[Ionic Framework](https://ionicframework.com/docs/intro) is an open source UI toolkit for building performant, high-quality mobile and desktop apps using web technologies (HTML, CSS, and JavaScript).


## Step 1: Set Up Your Ionic React Project

Set up an Ionic React app with [`npx`](https://www.npmjs.com/package/npx) and move into the project directory.

<Snippet dark text="npx ionic start [project-name] conference --type react && cd [project-name]" />
<Caption>Initializing an Ionic app with <Link href="https://www.npmjs.com/package/npx">npx</Link> and moving into the project directory.</Caption>

## Step 2: Deploying Your Ionic React App with ZEIT Now

With an Ionic app set up, it is ready to deploy live with [Now](/docs/v2).

Using [Now CLI](/download), deploy your app with **a single command**:

<Snippet dark text="now" />
<Caption>Deploying your Ionic app with Now CLI.</Caption>

If you want to deploy your Ionic app when you push to a Git repository, you can use either [Now for GitHub](/docs/v2/more/now-for-github), [Now for GitLab](/docs/v2/more/now-for-gitlab), or [Now for Bitbucket](/docs/v2/more/now-for-bitbucket) to have your project automatically deployed on every push, and the [production domain](https://zeit.co/docs/v2/custom-domains/#adding-a-domain) updated on push to master.

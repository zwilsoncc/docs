#### Application Error

The request for a Serverless Function reached the timeout threshold (5 minutes for paid accounts, 10 seconds for Hobby accounts). This is an application error.

<br />

<Note label={false}>
  <b>502</b>: Bad Gateway
  <br />
  Code: <InlineCode>FUNCTION_INVOCATION_TIMEOUT</InlineCode>
</Note>

---

#### Troubleshoot

- A common cause for this issue is when the application contains an unhandled exception. Check the application logs, which can be found at the host URL under [the `/_logs` path](/docs/v2/platform/deployments/#logs-view), for example:
  ```
  https://my-deployment-my-username.now.sh/_logs
  ```

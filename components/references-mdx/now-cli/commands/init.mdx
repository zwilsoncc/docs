## Init

The `now init` command is used to initialize projects locally from the examples found in the [<ProductName /> repository](https://github.com/zeit/now/tree/master/examples).

### Basic Usage

<Snippet dark text="now init" />
<Caption>Using the <InlineCode>now init</InlineCode> command to initialize a project locally.</Caption>

### Extended Usage

<Snippet dark text="now init [project-name]" />
<Caption>Using the <InlineCode>now init</InlineCode> command to initialize a specific project locally.</Caption>

<Snippet dark text="now init [project-name] [new-project-name]" />
<Caption>Using the <InlineCode>now init</InlineCode> command to initialize a specific project locally and rename the directory.</Caption>

### Global Options

The following [global options](#options) can be passed when using the `now init` command:

- `--debug`
- `--help`
- `--force`

For more information on global options and their usage, refer to the [options section](#options).
